Introduction

This document describes an application profile, in this case DCAT-AP. The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

This application profile has the status Ontwerp Standaard publised at 2022-09-13.

Information about the process and the decisions involved in the creation of this specification are consultable at the registry of standards.

License

This specification of ORGANISATION is published under "CC0".

Conformance

An implementation is conformant with this application profile if it satisfies these rules.

This document describes the usage of the following entities for a correct usage of the Application Profile:
